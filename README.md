# infosec-toolkit

A curated collection of tools and labs for studying and practicing information security (Infosec).

Focused on hands-on learning including network analysis, vulnerability testing, and security research.


# Notice

All files in this folder are provided for **educational and defensive security research purposes only**.  
Any unauthorized attack against real-world systems is **illegal and strictly prohibited**.  
Use these examples only in **personal labs or properly authorized testing environments**.
