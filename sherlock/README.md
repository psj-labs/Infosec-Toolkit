### OSINT (Open Source Intelligence) is an intelligence-gathering method that collects and analyzes publicly available information.

# Sherlock OSINT

### OSINT(Open Source Intelligence)는 누구나 접근 가능한 공개 정보를 수집·분석하는 정보 수집 기법입니다.
- Sherlock을 사용해 특정 유저네임이 여러 웹 서비스에 존재하는지 확인할 수 있습니다.
