# Inforsec-Toolkit
### This repository is a collection of tools used for penetration testing purposes.
### Each tool includes a PoC file that explains its functionality and usage.
### Unauthorized use against systems or websites without permission is illegal.

## 한국어 원문
### 본 리포지토리는 모의해킹 목적의 도구들을 정리한 저장소입니다.
### 각 도구에는 기능과 사용법을 설명하는 PoC 파일이 포함되어 있으며
### 사전 허가 없이 시스템 또는 웹사이트를 대상으로 사용하는 행위는 불법입니다.
